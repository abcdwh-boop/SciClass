1. Gemini Canvas란?

Gemini Canvas(제미나이 캔버스)는 단순히 채팅을 주고받는 단계를 넘어, 실시간으로 문서나 코드를 편집하고 미리보기를 제공하는 협업 환경입니다.

  • 대화형 인터페이스와 문서 편집기가 결합된 하이브리드 작업 공간
  • 프로그래밍 지식 없이도 웹 앱, 퀴즈 도구, 시뮬레이션 제작 가능
  • 실시간 피드백 및 부분 수정 기능을 통한 수업 자료 최적화

공통 방법

  1. Gemini 채팅창에 제작하고 싶은 도구를 요청합니다.
  2. 오른쪽 Canvas 창에 생성된 내용을 확인합니다.
  3. 상단의 '미리보기(Preview)' 버튼을 눌러 실제 작동을 테스트합니다.
  4. 수정이 필요하면 채팅으로 피드백을 주고받으며 완성합니다.

제작한 도구 공유하기(1) - 개인 계정일 때 가능

개인 구글 계정 사용 시 링크 하나로 작업 화면과 결과를 동시에 공유할 수 있는 가장 빠른 방법입니다.

1단계: 공유 버튼

Gemini 우측 상단의 '공유(Share)' 아이콘을 클릭합니다.

2단계: 링크 생성

'공유 링크 만들기' 버튼을 눌러 고유 주소를 생성합니다.

3단계: 주소 전달

복사된 링크를 동료 교사에게 전달하세요.

참고 안내: 학교(워크스페이스) 계정은 보안 정책상 '공유 링크 만들기' 버튼이 활성화되지 않을 수 있습니다. 이 경우 코드만 복사하여 전달하거나 아래의 웹 공유 방법을 활용하세요.

제작한 도구 공유하기(2) - 웹으로 공유하기

학교 계정 제약 없이, 학생들에게 깔끔한 웹 사이트 형태로 도구를 배포할 때 추천하는 방법입니다.

방법 A. GitHub

  • 1. index.html로 저장
  • 2. GitHub에 새 저장소 만들기
  • 3. 업로드 후 Pages 설정 활성화

방법 B. Netlify (가장 간편함)

  • 1. index.html을 폴더에 넣기
  • 2. Netlify Drop 사이트 접속
  • 3. 폴더를 화면으로 드래그 앤 드롭
